# WeddingCar Vietnam dichvuxecuoi.net

Nền tảng **kết nối dịch vụ xe cưới tại Việt Nam**.
Dự án cung cấp một **landing page sang trọng, tối ưu hiệu năng và trải nghiệm người dùng**, hướng tới việc phát triển thành **marketplace đặt xe cưới trực tuyến**.

Website cho phép:

* Giới thiệu các dòng **xe cưới cao cấp**
* Tìm kiếm và so sánh dịch vụ
* Đặt xe cưới cho ngày trọng đại

# Core Web Vitals Optimization

Trang web được tối ưu cho các chỉ số hiệu năng quan trọng:

### LCP (Largest Contentful Paint)

* Hero section nhẹ
* Font preload
* Hình ảnh tối ưu

### CLS (Cumulative Layout Shift)

* Layout cố định
* Image có kích thước ổn định

### INP / TBT

* Không có JavaScript nặng
* HTML/CSS thuần
